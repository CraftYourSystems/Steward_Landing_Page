# Purpose

The Footer provides visitors with essential navigation, company information, legal resources, and contact details.

It concludes the homepage with clarity and confidence.

Rather than acting as a secondary marketing section, it serves as the foundation beneath the Steward experience.

---

# User Question

"Where can I go next?"

---

# Business Goal

- Provide supporting navigation.
- Improve discoverability.
- Reinforce legitimacy.
- Offer clear contact information.
- Complete the homepage gracefully.

---

# Core Message

Steward is built with care, and every part of the experience—including the end of the page—should reflect that.

---

# Visual Style

Editorial.

Minimal.

Muted colors.

Soft divider.

Small typography.

Comfortable spacing.

No decorative graphics.

No background illustrations.

---

# Motion

None required.

If revealed on scroll:

Gentle fade.

No animations beyond page consistency.

The Footer should feel grounded and still.

---

# Interaction

Links

Subtle underline or opacity change on hover.

No dramatic transitions.

External links

Clearly indicated where appropriate.

Focus

Visible keyboard focus.

---

# Accessibility

Semantic <footer> element.

Logical heading hierarchy.

Keyboard accessible links.

Readable color contrast.

Meaningful link text.

---

# Copy Principles

Keep content concise.

Avoid promotional messaging.

Avoid marketing slogans.

Provide information—not persuasion.

---

# Voice

Calm.

Professional.

Reliable.

Simple.

The homepage has already made its case.

The Footer simply supports the visitor.

---

# Engineering Notes

Reusable Footer component.

Responsive grid.

Configurable navigation links.

Configurable contact details.

Dynamic copyright year.

No hardcoded URLs.

Support future social links without layout changes.

---

# Acceptance Criteria

Visitors should be able to:

- Navigate to key pages.
- Find contact information.
- Access legal documents.
- Understand who Steward is.
- Leave the homepage without confusion.

The Footer should feel complete but unobtrusive.

---

# Out of Scope

Newsletter signup.

Promotional banners.

Social media feeds.

Recent blog posts.

Live chat widgets.

Advertising.

Popups.

These intentionally remain outside Version 1.

---

# Definition of Done

The Footer is complete when it quietly answers every remaining navigational question without drawing attention to itself.

The homepage should end with the same clarity and confidence with which it began.
